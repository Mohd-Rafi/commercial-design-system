Loading indicators signal a user action is processing. Unlike progress bars, loading indicators only signal that loading is occurring and do not give any indication of progress. If a process will take more than a moment or two to complete, use a progress indicator instead.

Spinner Continuous

A spinner continuously rotates, indicating ongoing activity or loading progress. It serves as a visual cue, providing feedback during waiting periods and informing users that a process is taking place in the background.

Spinner Progress

Spinner progress visually indicates task completion, with a circular shape filling up or rotating. It provides feedback on tasks and estimates remaining time, enhancing the user experience.

Bar Continuous

A continuous bar visually represents ongoing activity or progress. It fills or moves across a horizontal or vertical bar, indicating that a process is in progress. It serves as a visual indicator, providing feedback to users during tasks or operations.

Bar Progress

A progress bar visually represents the completion of a task or process. It fills or moves across a horizontal or vertical bar, indicating the progress made. It provides visual feedback and allows users to estimate the remaining work or time.

Full-screen loading

The loading component should be used when the entire page is processing. This is often applied after data is submitted or saved by the user. If the process is going
to take more than a few minutes, consider including a notification.

When to use

  • An action temporarily disables the application

    If a user’s action temporarily disables the application while processing occurs, use a loading indicator and a full-screen overlay.

  • Data from user input is being saved or submitted

    Saving data following with user input often takes a few moments. Using a loading indicator allows the application to finish processing before allowing the user to continue.

  • Processing takes more than a few seconds

    If a loading process will require the application’s full resources and will take longer than a moment or two, use a full-screen loading indicator.

Loading spinners

Loading spinners may be scaled down if the loading experience is contextual to a certain item on the page.
