Search

Download More View Figma File

Search is an essential pattern for navigation or discovery. We live in the age of search, where a search engine is often the primary entry point into the internet. User expectations are high for the way search should work, and consistency is critical.

Search Stages

Search inputs come in different states depending on how the user is interacting with it — enabled, hover, focus, active, and filled.

Variations

There are two key variations of search — header search and independent search. Either variation can be collapsed or open by default.
